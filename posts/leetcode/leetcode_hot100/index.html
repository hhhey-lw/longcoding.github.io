1143. 最长公共子序列#

输入：text1 = "abcde", text2 = "ace" 
输出：3  
解释：最长公共子序列是 "ace" ，它的长度为 3 。

子序列，相对位置保持不变就行，不要求连续

dp[i][j] 含义， (0 - i-1)和(0 - j-1)的数组的最长公共子序列

转移方程：

​ if nums1[i-1] == nums2[j-1]:

​ dp[i][j] = dp[i-1][j-1] + 1 // 在之前的基础上最长子数组+1

else:

​ dp[i][j] = Max(dp[i-1][j], dp[i][j-1]) // 不要求连续
